css的引入方式

css有三种引入方式，分别是：

  • 行间样式（优先级别是最高的，一般不建议这么使用）
  • 内部引入（在header部分用下列代码引入）
<style type="text/css">...</style>
  • 外部引入（在header部分用下列代码引入）
<link rel='stylesheet' type="text/css" href="css的路径"/>
